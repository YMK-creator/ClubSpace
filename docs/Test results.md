| ---------------------------------- | ------------------------------------------------------------------------ | ------------------------------------------------------------------------ | ----------------------------------- | ------------ |
| **UC1: Регистрация**               |                                                                          |                                                                          |                                     |              |
| 001-1: Успешная регистрация        | Заполнить форму (Username, Email, Password, Confirm) и нажать "Register" | Учетная запись создана (201 Created), редирект на вход или дашборд       | Регистрация проходит успешно        | Тест пройден |

| 007-3: Сохранение сессии           | Обновить страницу после входа                                            | Пользователь остается авторизован                                        | Состояние авторизации сохраняется   | Тест пройден |

Все основные сценарии использования протестированы.
